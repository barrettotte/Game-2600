# Notes


## Playfield
* PF0 - upper 4 bits used, reverse order output
* PF1 - 8 bits, normal output
* PF2 - 8 bits, reverse order output



## Memory Map
```
+---------------------------+
|       TIA Registers       | $00-$7F
+---------------------------+
|         PIA RAM           | $80-$FF
+---------------------------+
|           ???             |
+---------------------------+
|    PIA Ports and Timer    | $280-$297
+---------------------------+
|           ???             |
+---------------------------+
|       Cartridge ROM       | $F000-$FFFF
+---------------------------+
```



## NTSC Video Frame
```

+-------------------------------------------------+        <-----+
|                 Vertical Sync                   |   3 lines    |
|                                                 |              |
+-------------------------------------------------+              |
|                 Vertical Blank                  |  37 lines    |
|                                                 |              |
+--------------------+----------------------------+              +-- 262 lines
|    Horiz.          |           Visible          |              |
|    Blank           |            Frame           | 192 lines    |
|                    |  <-- 160 color clocks -->  |              |
+--------------------+----------------------------+              |
|                    Overscan                     |  30 lines    |
|                                                 |              |
+-------------------------------------------------+        <-----+

|--------------  228 color clocks ----------------|
|---------------  76  CPU cycles -----------------|

```
